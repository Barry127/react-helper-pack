Getting Started

Welcome

react-helper-pack is a collection of helper functions to create your own React components. I was copy-pasting a lot of code between different React projects, this collection is a set of reusable code to keep my development DRY.

Installation

The easiest way to install react-helper-pack is using npm and import the functions to your files.

npm install --save react-helper-pack
// Load everything
import * as reactHelpers from 'react-helper-pack';

// Load specific methods
import { filterEvents } from 'react-helper-pack';

Clone the repo to run tests.

git clone https://github.com/Barry127/react-helper-pack.git
cd react-helper-pack
npm run test

Methods

filterEvents

filterEvents(props, [ignore])

Takes a set of props and filter it to only return the events.

Arguments

  • props (Object): The props object to filter from.
  • [ignore] (Array|String): events to ignore (to not return).

Returns

  • (Object): Object only containing events.

Example

import { filterEvents } from 'react-helper-pack';

const props = {
  disabled: true,
  someKey: 'someValue',
  onClick: () => console.log('click'),
  onMouseDown () => console.log('mousedown')
};

filterProps(props);
// => { onClick: [Function], onMouseDown: [Function] }

filterProps(props, ['onMouseDown']);
// => { onClick: [Function] }
